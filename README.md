## 特点

- 无Logo、免登录。
- 每次只加载一个项目、一个填写步骤和一个样品，适合较旧平板。
- 覆盖委托样品信息、环境、设备、参数、过程确认、原始数据、辅助数据、异常偏离、附件归档和人员信息。
- 自动计算和判定。
- 可下载JSON草稿，并重新导入继续填写。
- Word输出直接使用 `templates/` 中的CMA原始记录模板。
