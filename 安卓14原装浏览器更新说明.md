# Android 14 原装浏览器优化版

## 已优化
- 取消页面动画、进度条和无关工具栏。
- 单次只加载一个实验步骤、一个试样。
- 输入框固定为16px，避免平板点击后自动放大。
- 按钮高度不小于48px，便于触控。
- 禁止页面横向溢出，表单自动适应窄屏。
- Word组件延迟到导出步骤才加载，缩短首页等待时间。
- 上传文件限制为6MB，减少低配置平板内存压力。

## 更新现有GitHub
1. 解压本ZIP。
2. 打开原GitHub仓库。
3. Add file → Upload files。
4. 上传解压后全部同名文件，覆盖旧文件。
5. Commit changes。
6. 等待Streamlit自动更新；如未更新，Manage app → Reboot app。
7. 平板彻底关闭旧网页，再重新打开；必要时清除网站缓存。
